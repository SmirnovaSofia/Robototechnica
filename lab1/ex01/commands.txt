1. nc (netcat) - Эта команда позволяет устанавливать сетевые соединения, сканировать порты, передавать файлы и данные через сеть.

2. fdupes - Используется для поиска и удаления дублирующихся файлов в системе.

3. lsof (List Open Files) - Команда показывает список всех открытых файлов и процессов, которые их используют.

4. ss (socket statistics) - Предоставляет информацию о сокетах, заменяет команду netstat.

5. strace - Используется для отслеживания системных вызовов и сигналов процесса.

6. tac - Выводит содержимое файла в обратном порядке, начиная с последней строки.

7. chage - Управляет сроком действия пароля пользователя в системе.

8. timeout - Запускает команду с ограничением по времени выполнения.

9. pv (pipe viewer) - Показывает прогресс передачи данных через канал (pipe).

10. rsync - Утилита для синхронизации и передачи файлов между машинами.
